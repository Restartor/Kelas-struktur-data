# Kelas Struktur Data

Repository ini berisi catatan dan contoh program untuk materi Struktur Data per pertemuan.

## Pertemuan 1
Penjelasan dasar array.

## Pertemuan 2
Input dan output pada array.

## Pertemuan 3
Penggunaan struct pada array.

## Pertemuan 4
Materi searching, berisi implementasi:

- Sequential Search untuk mencari data dengan pemeriksaan satu per satu.
- Binary Search untuk mencari data dengan pembagian rentang pencarian.

Program meminta jumlah data, menerima input array, lalu menampilkan hasil pencarian berdasarkan nilai yang dimasukkan pengguna.

## Pertemuan 5
Materi sorting, berisi implementasi Bubble Sort untuk mengurutkan data dari kecil ke besar.

Program menampilkan data sebelum diurutkan, melakukan pertukaran elemen jika urutannya salah, lalu menampilkan hasil akhir setelah proses sorting selesai.

## Cara Menjalankan
Gunakan compiler C++ seperti `g++` untuk menjalankan file pertemuan yang diinginkan.

Contoh:

```bash
g++ Pertemuan-4.cpp -o Pertemuan-4.exe
g++ Pertemuan-5.cpp -o Pertemuan-5.exe
```

## Ringkasan Materi
- Array
- Struct pada array
- Searching
- Sorting
